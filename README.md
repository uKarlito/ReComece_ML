Arquivos:
1. dados_recomece.csv: Dados simulados de perfis de carreira.
2. modelo_classificacao.ipynb: Notebook com treinamento do modelo de recomendação de trilhas (Random Forest).
3. modelo_clusterizacao.ipynb: Notebook com agrupamento social (K-Means).
4. app.py: API Flask que disponibiliza o modelo de classificação.
5. *.joblib: Arquivos binários dos modelos treinados.

Como rodar:
1. Instale as dependências: pip install pandas flask scikit-learn joblib
2. Execute app.py para iniciar a API.
