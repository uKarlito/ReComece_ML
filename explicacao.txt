SOBRE OS DADOS (DATASET)
==================================================
Nome do arquivo: dados_recomece.csv

Justificativa para Geração de Dados Sintéticos:
Optamos pela geração de dados sintéticos (fictícios) por três motivos principais:

1. "Cold Start" (Início Frio): Como o projeto ReComece é uma solução nova, ainda não possuímos uma base de usuários reais ou histórico de navegação para treinar os modelos.

2. Controle de Cenários: Gerar os dados nos permitiu criar cenários balanceados e regras específicas (ex: correlação entre gosto por matemática e Data Science) para validar se os algoritmos de IA conseguem aprender os padrões corretamente.

Como os dados foram gerados:
Utilizamos um script Python (gerar_dados.py) utilizando a biblioteca 'random' para criar 1.000 perfis simulados. Definimos regras lógicas de negócio para atribuir a 'trilha_recomendada' com base nos interesses (features), garantindo que houvesse um padrão detectável para o modelo supervisionado.

==================================================
MODELOS DE IA DESENVOLVIDOS
==================================================

1. Classificação (Recomendação de Trilha):
   - Algoritmo: RandomForestClassifier.
   - Objetivo: Receber as aptidões do usuário (ex: gosta de matemática e tem alto interesse técnico) e prever qual a melhor trilha de aprendizado (ex: Data Science).
   - Arquivo Notebook: modelo_classificacao.ipynb
   - Modelo Salvo: modelo_trilha.joblib

2. Agrupamento (Segmentação Social):
   - Algoritmo: K-Means.
   - Objetivo: Identificar grupos de vulnerabilidade social com base em Renda, Idade e Tempo de Desemprego para oferecer bolsas.
   - Arquivo Notebook: modelo_clusterizacao.ipynb
   - Modelo Salvo: modelo_cluster.joblib
